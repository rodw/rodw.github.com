Working with R in emacs

1) Install R.

2) Install ESS:

sudo aptitude install ess=12.09-1

3) Require ess-site:

(add-to-list 'load-path "/usr/share/emacs/site-lisp/")
(require 'ess-site)

4) Load R within emacs via M-x R.
