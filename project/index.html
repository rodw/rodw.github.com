CoffeeScript

I've enjoyed working with CoffeeScript (Node.js) recently. Here are some modules I've posted on GitHub.
  • named-parameters is an npm module that makes it easier to write JavaScript/CoffeeScript methods that accept maps as arguments (rather than arrays). Think node-optimist, but for function arguments rather than command line parameters.
  • phony is an npm module that generates realistic-looking fake names, addresses, text, email addresses, etc. for testing, demonstration or random inspiration.
  • tiny-node.js-webserver does what it says on the tin: a simple web server implemented in a single small CoffeeScript file.

The Axion RDMS

Once upon a time I created a fully-functional relational database from scratch (with a bit of help from collaborators). I was the primary developer behind Axion, a small, fast, open source relational database system written in and for Java. Axion has been dormant for quite a while, but it was and still is a fairly robust and complete system. Axion had and as far as I know still has some features no other RDBMS provides. Axion...

  • is embedded (it runs in-process within the same JVM as the client code),
  • supports both persitent (on-disk) and transient (in-memory) modes,
  • supports both ANSI SQL and several extensions from Oracle's SQL dialect (circa 10g or whatever was released at the time),
  • is fully transactional,
  • is fully ACID (providing Atomicity, Consistency and Isolation and Durability),
  • can be easily extended with custom "pluggable" table, index, column (field) and function types.

Building this wasn't as crazy or impractical as it might sound.

Brightspoke, a search engine for bikes.

The last time I was on the market for a new bicycle I created a tool for comparing bikes that I think is pretty nifty. The bike data is now stale, but the stem fit calculator remains popular. The calculator can help you determine how your handlebars will move when you change your stem. (The stem being the bit on a bike that connects the handlebars to the fork.)
